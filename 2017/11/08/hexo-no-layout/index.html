Hexo跳过渲染，创建自定义页面

发表于 2017-11-08 | 分类于 开发笔记 | 本文总阅读量 次

Hexo 博客中所生成的文章都是经由渲染的静态网页，而静态网页的样式都直接由 Hexo 的主题控制，所以 Hexo 博客大部分都呈现出一种高度的统一化与规范化。不过 Hexo 提供了跳过渲染功能，使得我们可以直接在博客中放入自定义网页。
比如前面博文有提到的自定义 404.html

一、创建自定义网页

网页可以是自己编写的，也可以是别人现成的源码，比如自定义网页为 test.html 。
网页编写完成后，在 Hexo\source 目录下创建一个文件夹（文件夹名称任意，比如我为了测试创建的是test这个文件夹。
将 html 文件放置于此文件夹。
