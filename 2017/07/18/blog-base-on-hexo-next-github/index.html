基于Hexo+NexT+GitHub搭建个人博客

发表于 2017-07-18 | 分类于 开发笔记
基于Hexo+NexT+GitHub搭建个人博客

Hexo

安装

中文官方网址：https://hexo.io/zh-cn/

安装Hexo的步骤非常简单，而且官网文档也非常详尽，官网文档地址：https://hexo.io/zh-cn/docs/

安装 Hexo 相当简单。然而在安装前，您必须检查电脑中是否已安装下列应用程序：

  • Node.js

如果您的电脑中已经安装上述必备程序，那么恭喜您！接下来只需要使用 npm 即可完成 Hexo 的安装。

$ npm install -g hexo-cli

完成上述操作Hexo就算安装完成了，非常简单。

建站

参见官方文档地址：https://hexo.io/zh-cn/docs/setup.html

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init <folder>
$ cd <folder>
 $ npm install

新建完成后，指定文件夹的目录如下：

.
├── _config.yml
├── package.json
├── scaffolds
├── source
| ├── _drafts
| └── _posts
└── themes

_config.yml 网站的配置信息，您可以在此配置大部分的参数。
关于配置官网也给出了非常详细的说明：https://hexo.io/zh-cn/docs/configuration.html

需要特别注意的点是： 属性设置后面的：需要有空格再填写内容！！！

安装NexT主题

Hexo是支持各种主题安装的，而NexT只是其中的一个，其他你可以到Hexo的主题库里去选自己喜欢的主题，因为我目前选用的就是NexT主题所以这里就着重介绍这个主题的安装。

NexT主题官网地址：http://theme-next.iissnan.com/

NexT的安装在官网上也有非常详细的说明，官方文档地址：http://theme-next.iissnan.com/getting-started.html

将主题文件下载下来之后，拷贝到站点目录下的 themes 目录下，然后再在站点的配置文件_config.yml中修改一下配置：

theme: next

即可。

GitHub

申请注册账号

GitHub官网：https://github.com
申请注册一个账号就可以了。
